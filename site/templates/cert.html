Certificate Installation

Install Security Certificate

To enable secure HTTPS interception, you must install the proxy's root certificate into your system's trust store. This is a one-time setup step.

Certificate Location:
{{ cert_path }}

Installation Instructions for Windows

1
Navigate to the certificate location shown above using File Explorer. Look for the file named mitmproxy-ca-cert.cer.
2
Double-click the mitmproxy-ca-cert.cer file to open the Certificate Import Wizard.
3
Click Install Certificate... button.
4
Select Store Location: Local Machine. Click Next.
(This requires administrator privileges)
5
Select "Place all certificates in the following store". Click Browse....
6
Choose Trusted Root Certification Authorities. Click OK, then Next.
7
Click Finish. You will see a security warning. Click Yes to proceed with the installation.
Note: After installing the certificate, you may need to restart your browser for the changes to take effect.
