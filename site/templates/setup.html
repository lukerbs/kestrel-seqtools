Proxy Configuration Setup

Quick Start

This application requires two setup steps:

  1. Configure your browser to use the local proxy
  2. Install the security certificate
Install Certificate First
Configure Browser Proxy
Option A: Automatic Configuration (PAC File)
1
Copy this PAC file URL:
{{ pac_url }}
2
In Chrome/Edge, go to: Settings → System → Open your computer's proxy settings
3
Enable "Use setup script" and paste the PAC file URL above
Option B: Manual Configuration
1
In Chrome/Edge, go to: Settings → System → Open your computer's proxy settings
2
Enable "Use a proxy server" and configure:
Address: {{ proxy_host }}
Port: {{ proxy_port }}
Install Security Certificate

After configuring the proxy, you must install the security certificate to allow HTTPS interception.

View Certificate Instructions
